# 小小文明 · Tiny Empires

A tiny, kid-friendly Age-of-Empires-style RTS — built for Sam to play with his son.
Pick a civilization (中国 / 希腊), gather 🌲🍒, build a barracks ⛺, train an army, raze the enemy 🏯.
Each civ has its own strengths — no "best" civ.

Single-file HTML5 Canvas. Zero build, zero dependencies. Open `index.html` or play online.

Design: see `DESIGN_v1.md`.
